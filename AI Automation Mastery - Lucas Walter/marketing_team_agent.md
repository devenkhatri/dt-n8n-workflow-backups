# Workflow Analysis for The Recap AI - Marketing Team Agent

## Description
An AI-powered marketing assistant that helps create and repurpose content across multiple channels—including newsletters, social media posts, short-form video scripts, and research reports—while maintaining brand voice and strategic alignment.

## Input Details
The workflow is triggered either by an incoming chat message or a webhook request containing a user’s marketing-related request.

## Process Summary
The agent uses a large language model (Gemini 2.5 Pro) with memory to understand and respond to user requests. It can generate daily newsletters, create custom images, repurpose newsletter content into Twitter threads, and produce short-form video scripts. It also supports deep research on topics using Perplexity AI and can email the resulting report. Additionally, it can generate talking-head videos using previously created scripts. Strategic planning is supported via a dedicated 'think' tool to structure complex tasks.

## Output Details
The workflow produces various marketing assets such as newsletters, social media content, video scripts, research reports, and images, which are either returned to the user via chat/webhook or delivered via email.
