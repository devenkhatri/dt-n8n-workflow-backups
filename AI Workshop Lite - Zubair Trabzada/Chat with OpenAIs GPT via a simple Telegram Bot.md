# Workflow Analysis for Telegram AI Chatbot

## Description
This workflow creates an AI chatbot within Telegram that responds to user messages using an OpenAI chat model.

## Input Details
The workflow is triggered by new messages received in a Telegram chat.

## Process Summary
Upon receiving a message in Telegram, the workflow forwards the message text to an AI Agent. The AI Agent utilizes an OpenAI Chat Model to generate a helpful response, incorporating emojis as instructed. Finally, the generated AI response is sent back to the original Telegram chat.

## Output Details
The workflow sends an AI-generated text response back to the Telegram chat.
