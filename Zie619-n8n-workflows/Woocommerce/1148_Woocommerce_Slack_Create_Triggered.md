# Workflow Analysis for New WooCommerce product to Slack

## Description
This workflow automatically notifies a Slack channel whenever a new product is added to your WooCommerce store.

## Input Details
This workflow is triggered by a webhook whenever a new product is created in WooCommerce, receiving detailed product information.

## Process Summary
First, the workflow is activated when a new product is created in WooCommerce. Next, it gathers the product's name, price, sale price, permalink, and creation date. Finally, it constructs a message and sends it to the "woo-commerce" Slack channel, announcing the new product with its details.

## Output Details
The workflow sends a formatted message containing new product details to a designated Slack channel.
