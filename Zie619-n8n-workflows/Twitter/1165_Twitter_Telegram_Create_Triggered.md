# Workflow Analysis for New WooCommerce Product to Twitter and Telegram

## Description
This automated workflow processes data and performs automated tasks by announcing new WooCommerce products on Twitter and Telegram.

## Input Details
This workflow is triggered by a webhook when a new product is created in WooCommerce, receiving details about the newly added product.

## Process Summary
When a new product is created in WooCommerce, the workflow captures the product details. It then constructs an announcement message using the product's name and permalink. This announcement is simultaneously posted to Twitter and sent to a specified Telegram chat. An error handler is in place to manage any execution failures.

## Output Details
The workflow posts new product announcements to Twitter and sends messages to a Telegram chat.
