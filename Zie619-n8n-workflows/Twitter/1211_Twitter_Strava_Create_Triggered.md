# Workflow Analysis for Receive updates when a new activity gets created and tweet about it

## Description
This workflow automates the process of tweeting about new activities created on Strava.

## Input Details
The workflow is triggered by new activity creation events from Strava.

## Process Summary
First, the workflow is activated when a new activity is created on Strava. Next, it extracts details like distance and activity name from the Strava activity data. Finally, it composes a tweet using this information and posts it to Twitter.

## Output Details
The workflow posts a new tweet to Twitter with details about the Strava activity.
