# Workflow Analysis for Mattermost Workflow

## Description
Automated workflow: Mattermost Workflow. This workflow processes data and performs automated tasks.

## Input Details
The workflow is triggered by an Emelia webhook when a contact replies to a specific campaign.

## Process Summary
The workflow starts when a contact replies to a campaign in Emelia. It then constructs a notification message with the contact's first name and company. Finally, this message is sent to a predefined Mattermost channel.

## Output Details
The workflow sends a notification message to a Mattermost channel.
