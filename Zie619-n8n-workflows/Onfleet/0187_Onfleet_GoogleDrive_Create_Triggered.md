# Workflow Analysis for Create an Onfleet task when a file in Google Drive is updated

## Description
Automated workflow: Create an Onfleet task when a file in Google Drive is updated. This workflow processes data and performs automated tasks.

## Input Details
This workflow is triggered every minute when a specific file in Google Drive is updated.

## Process Summary
The workflow continuously monitors a specified Google Drive file for updates. Upon detecting an update, it proceeds to create a new task in Onfleet. An error handler is in place to catch and manage any execution failures, stopping the workflow with a specified error message.

## Output Details
The workflow creates a new task in Onfleet.
