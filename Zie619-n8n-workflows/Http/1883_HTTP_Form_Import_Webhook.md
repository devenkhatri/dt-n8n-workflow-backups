# Workflow Analysis for Get PDF with JSReport

## Description
This workflow automates the creation and delivery of invoices by capturing billing information, generating a PDF invoice using JSReport, and then emailing it to the recipient.

## Input Details
The workflow is triggered by a form submission, which collects essential billing details such as buyer information and item specifics.

## Process Summary
First, the workflow gathers invoice details through a user-friendly form. Next, it sends this collected data to the JSReport API to dynamically generate an invoice document, specifically a PDF. Finally, the generated PDF invoice is attached to an email and automatically sent to a designated recipient.

## Output Details
The workflow produces a PDF invoice and sends it as an email attachment via Gmail to "contact@nonocode.fr".
