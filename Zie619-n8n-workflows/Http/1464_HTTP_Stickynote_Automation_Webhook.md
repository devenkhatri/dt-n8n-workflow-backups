# Workflow Analysis for AI-Powered Research with Jina AI Deep Search

## Description
This fully automated workflow leverages Jina AI’s DeepSearch to conduct advanced AI-powered research and generate structured, fact-based reports without requiring an API key. It democratizes automated knowledge discovery by making it freely accessible.

## Input Details
The workflow is triggered manually or by a user entering a research query through a chat interface.

## Process Summary
1. A user submits a research query via a chat interface. 2. The workflow sends this query to Jina AI’s DeepSearch API for in-depth analysis. 3. Jina AI DeepSearch retrieves and analyzes relevant information, providing key insights, structured analysis, and sources. 4. A subsequent code node processes the AI's response, extracting, cleaning, and formatting the insights into a well-structured Markdown report, ensuring proper URL and footnote presentation.

## Output Details
The workflow produces a final, well-structured research report in Markdown format, ready for immediate use.
