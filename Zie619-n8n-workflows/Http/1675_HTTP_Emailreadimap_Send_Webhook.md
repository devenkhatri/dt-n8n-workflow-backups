# Workflow Analysis for Summarize emails with A.I. then send to messenger

## Description
This automated workflow summarizes emails using artificial intelligence and then sends the summaries to a messenger application. It integrates email reading, AI processing, and messaging services.

## Input Details
This workflow is manually triggered and then reads emails from a configured IMAP email account.

## Process Summary
The workflow begins with a manual trigger. It then proceeds to read emails from an IMAP server. The content of each email, including sender, subject, and HTML body, is sent to an AI model for summarization, with specific instructions to highlight important details, action items, and deadlines. Finally, the AI-generated summary is sent as a text message to a designated messenger platform.

## Output Details
The workflow produces AI-summarized email content which is then sent as a text message to a messenger platform.
