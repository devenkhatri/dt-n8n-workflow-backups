# Workflow Analysis for Twitter Workflow

## Description
Automated workflow: Twitter Workflow. This workflow processes data and performs automated tasks.

## Input Details
This workflow is triggered whenever a new product is created in Shopify and receives product details as input.

## Process Summary
Upon a new product creation in Shopify, the workflow extracts product details such as vendor, title, and product type. It then simultaneously composes and sends a promotional message to Twitter. In parallel, it composes and sends a similar promotional message to a specified Telegram chat.

## Output Details
The workflow publishes promotional messages on Twitter and to a Telegram chat.
