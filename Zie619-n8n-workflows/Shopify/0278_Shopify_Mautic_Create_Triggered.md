# Workflow Analysis for Shopifytrigger Workflow

## Description
This workflow automates the process of synchronizing new customer data from Shopify to Mautic, ensuring that your marketing platform is always up-to-date with your customer base.

## Input Details
This workflow is triggered automatically when a new customer is created in Shopify and receives the new customer's details as input.

## Process Summary
First, the workflow is activated upon the creation of a new customer in Shopify. Next, it extracts the new customer's first name, last name, and email address. Finally, it uses this information to create a new contact in Mautic.

## Output Details
The workflow creates a new contact in Mautic, populating it with the new customer's first name, last name, and email.
